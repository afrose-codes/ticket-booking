# Ticket Booking System 🎟️

## Description
This is a React-based web application for booking tickets for department events.

## Features
- Login system (Admin/User)
- Event selection (Tech Fest, Seminar)
- Ticket booking
- MySQL database integration
- Admin dashboard

## Tech Stack
- React JS
- Node.js (Express)
- MySQL

## How to Run
1. npm install
2. npm start
3. Run backend using node server.js
